heroku logs - логи
heroku login - вход в хероки
heroku create - создание приложения
heroku push heroku master - отправляем файлы на хероку
heroku ps:scale worker=1 - запуск программы на хосте
heroku buildpacks:clear - 
heroku git:clone -a 'имя приложения на хероки' - клонирует с Хероки на PC
heroku git:remote -a 'имя приложения на хероки' - добавляет GIT в приложение
